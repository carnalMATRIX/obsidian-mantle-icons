# Mantle Icons

> [!CAUTION]
> **Status: Beta (v1.0.0)**  
> This plugin is currently in Alpha. Features and UI are subject to change.

Mantle Icons provides seamless icon integration for your Obsidian notes and UI. It is designed to work as a companion to the Zenith theme, providing a consistent visual language across your vault.

## Features
- **Icon Suggester**: Easily insert icons into your notes using a dedicated suggester.
- **Lucide Integration**: Leverages the comprehensive Lucide icon set.
- **Zenith Optimized**: Specifically tuned to match the spacing and color hierarchy of the [Zenith theme](https://github.com/carnalMATRIX/obsidian-mantle-zenith).

## Installation

### Manual Installation
1. Download the `main.js`, `manifest.json`, and `styles.css` from the latest release.
2. Create a folder named `mantle-icons` in your vault's `.obsidian/plugins/` directory.
3. Move the downloaded files into that folder.
4. Restart Obsidian and enable **Mantle Icons** in **Settings > Community plugins**.

## Development

To modify this plugin:
1. Navigate to this directory in your terminal.
2. Install dependencies: `npm install`
3. Build the plugin: `npm run build`
4. For active development, use: `npm run dev`

This plugin is built with TypeScript and esbuild.
